Introducción

El proyecto final de la materia consiste en realizar la codificación de un dashboard capaz de leer sensores enviados desde un dispositivo NodeMCU además de validar el ingreso mediante un login para poder visualizar el dashboard. Esto permite juntar los conocimientos adquiridos en la materia de Redes Digitales de datos y en su laboratorio. Se usaron tecnologías como HTML5, CSS3, JS, PHP y C para realizar los códigos necesarios en la creación de la página web tanto como del programa para hacer la lectura de los sensores mediante el dispositivo ESP8266.

Objetivos

Objetivo General

Creación de una página web usando PHP para almacenar los valores de alguno sensores conectados a dispositivos NodeMCU

Objetivos específicos

  1. Creación de la página PHP
  2. Realización del dódigo para al NodeMCU
  3. Creación de tablero para registro y muestra de las incidencias de los sensores
  4. Probar los códigos desde tres dispositivos NodeMCU

Marco Teórico

PHP es un lenguaje de programación de uso general que se adapta especialmente al desarrollo web.​ Fue creado inicialmente por el programador danés-canadiense Rasmus Lerdorf en 1994.​ En la actualidad, la implementación de referencia de PHP es producida por The PHP Group

JavaScript es un lenguaje de programación interpretado, dialecto del estándar ECMAScript. Se define como orientado a objetos, ​ basado en prototipos, imperativo, débilmente tipado y dinámico.

C es un lenguaje de programación de propósito general​ originalmente desarrollado por Dennis Ritchie entre 1969 y 1972 en los Laboratorios Bell, ​ como evolución del anterior lenguaje B, a su vez basado en BCPL.​​​Al igual que B, es un lenguaje orientado a la implementación de sistemas operativos, concretamente Unix.

HTML 5 es la quinta revisión importante del lenguaje básico de la World Wide Web, HTML. HTML5 específica dos variantes de sintaxis para HTML: una «clásica», HTML, conocida como HTML5, y una variante XHTML conocida como sintaxis XHTML 5 que deberá servirse con sintaxis XML.​​

CSS, en español «Hojas de estilo en cascada», es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado.​

NodeMCU es una plataforma IoT de código abierto.​​ Incluye el firmware que se ejecuta en el SoC Wi-Fi ESP8266 de Espressif Systems y el hardware que se basa en el módulo ESP-12.​​ El término "NodeMCU" se refiere al firmware en lugar de a los kits de desarrollo. El firmware utiliza el lenguaje Lua.
